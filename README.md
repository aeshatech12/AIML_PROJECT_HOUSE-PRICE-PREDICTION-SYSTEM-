# AI/ML House Price Prediction System

This project is a Machine Learning based House Price Prediction System that predicts the estimated price of a house based on various input features such as area, number of bedrooms, bathrooms, location, and other important factors.

The system uses data preprocessing, model training, and prediction techniques to provide accurate house price estimates. It helps users, buyers, sellers, and real estate businesses understand approximate property values using historical housing data.

## Features

- Predict house prices instantly  
- User-friendly interface  
- Trained Machine Learning model  
- Data visualization of housing trends  
- Accurate price estimation based on inputs  
- Easy to use and efficient system  

## Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  
- Tkinter / Streamlit (if GUI used)

## Machine Learning Techniques Used

- Data Cleaning  
- Feature Selection  
- Train-Test Split  
- Linear Regression / Random Forest Regression  
- Model Evaluation using RMSE and R² Score  

## How to Run

1. Clone or download this repository  
2. Install required libraries  
3. Run the main Python file  
4. Enter house details  
5. Get predicted house price  

## Objective

To build an intelligent system that predicts house prices using Machine Learning algorithms and improves decision-making in real estate.
